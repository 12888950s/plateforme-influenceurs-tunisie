# Plateforme de classification des influenceurs

## Description
Ce projet utilise le Machine Learning pour classifier les influenceurs à partir de données issues des réseaux sociaux.

## Fonctionnalités
- Web scraping des données
- Prétraitement des données
- Modélisation Machine Learning
- Dashboard web

## Technologies
- Python
- Pandas, Scikit-learn
- Jupyter Notebook

## Exécution
```bash
pip install -r requirements.txt
python dashboard_web.py
